Problem to be addressed

  • Key word identification

    • This function mainly focus on identifying the key words of a provided mathematical word problem in Sinhala language. In solving this problem Part Of Speech tagging(POS tagging) techniques has been applied.

  • Mathematical Problem Identification

    • This function is focused on meaning generation. The target is to identify what the problem is based on. System looks for 'wh questions' and identifies to which question category the provided problem falls in to. this step involves training a model to identify question pattern of a sentence.For that Conditional Random Field (CRF) algorithm has been used

  • Mathematical Operation Identification

    • This research part is involved in identifying the correct mathematical operation for a given word problem. In this approach the verb categorization has been applied. There we developed a method to find mathematical operation using relevant keywords and verb categories.After finding the consisting of entities (E), Containers (C), Attributes (A), relations(R) among entities and quantities, form the states < s1, s2… sT > by updating quantities in every container using learned verb categories.After finding the relevant mathematical operation ,form the equation by comparing the quantities for matched containers between two states.identification, mathematical operation and the output of sentence classification are combined together in neural network to solve the problem and generate the correct answer.
